¿Que es la programacion?

La programación es el proceso de crear un conjunto de instrucciones que le dicen a una computadora como realizar algún tipo de tarea. Pero no solo la acción de escribir un código para que la computadora o el software lo ejecute. Incluye, además, todas las tareas necesarias para que el código funcione correctamente y cumpla el objetivo para el cual se escribió. En la actualidad, la noción de programación se encuentra muy asociada a la creación de aplicaciones de informática y videojuegos.
